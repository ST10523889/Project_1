Kleancutz is a private barbering organization that has recently been opened with the mission of introducing affordable private barbering services as all people need haircut maintenance but may struggle due to the increased prices in the industry. 

Website objectives and goals:

Almost all successful organizations have something in common – a website. Therefore, if Kleancutz aims to be among one of them, it has to act as one. The purpose of the website would be to showcase the work of the business, attract leads and incorporate a way for customers to schedule a service with the business.

Essential website features:

1 - A home page which should be short, concise, showcase images of what the business does and concisely explain the business. Also contain buttons to navigate to other pages.

2 - The services page should be the first page button showcased and also should be on the home page. It should show all the business’ service offerings, mainly through images and also display the price or price range. Most importantly it should include some sort of booking/scheduling system where customers can book a service, and it should be super easy for all to operate. 

3 - The enquiry page which most importantly should include some sort of booking/scheduling system where customers can book a service, and it should be super easy for all to operate.

4 - An about us page to fully explain how the business started, with images to back it up, and talk about the vision, mission, and journey of the organization.

5 - The contact page should include the organization’s contact details and allow customers  to reach out to the business.

Timeline and milestones:

The aim is to get the project done in 3 parts, with the first one being done before the end of the 20th of April. The first part is basically laying the foundation of the project with the 2nd and third being the “real work”. Several aspects about building websites will be learnt throughout the project.
